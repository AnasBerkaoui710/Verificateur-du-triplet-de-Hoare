Language Syntax & Rules Documentation

Arithmetic Expressions

  • Addition: x + 1
  • Subtraction: y - 5
  • Multiplication: z * 3

  • Modulo: y % 2 (remainder)
  • Grouping: (x + y) * z

Boolean Conditions

  • Comparisons: ==, !=, <, <=, >, >=
  • Operators: and, or, not
  • Constants: true, false
